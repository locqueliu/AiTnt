# AiTnt

[English Version](./README.md)

AiTnt 是我自己搭的一套本地桌面创作工作站，用来把图像生成、视频生成、可复用素材、节点工作流和高频场景工具放进同一个应用里。

我不太喜欢把创作流程拆散在很多文件夹、网页和零碎工具之间，所以 AiTnt 的出发点一直很明确：让提示词、输出结果、工作流和模型配置在同一个本地工作区里连起来。

## 当前能力

- 图像工作区：支持文生图、图生图和本地输出管理
- 视频工作区：支持生成任务、队列式运行和导出流程
- 资源库：集中管理提示词、模板和可复用创作素材
- 节点画布：支持工作流导入、导出和结构整理
- 快捷应用：面向商品图、风格化等高频生产场景
- 设置中心：统一管理供应商、模型、目录、语言和工作区偏好

## 技术栈

- Electron
- React
- TypeScript
- Vite
- Zustand
- XYFlow
- dnd-kit

## 本地开发

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

构建桌面应用：

```bash
npm run build
```

## 项目思路

AiTnt 一直围绕这几件事来做：

- 让生成和整理发生在同一个工作区里
- 让能复用的工作流更容易沉淀下来
- 让本地创作环境打开快、续做也顺手
- 让视觉输出、提示词资产和节点逻辑同等重要

## 目录说明

- `src/` 应用源码
- `scripts/` 构建辅助与准备脚本
- `build/` 打包资源与应用素材
- `public/` 应用使用的静态资源
