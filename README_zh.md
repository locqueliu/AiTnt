# AiTnt

[English Version](./README.md)

AiTnt 是我自己一直在打磨的一套本地桌面创作工作站。

一开始做它，其实就是因为我不想把图像生成、视频生成、提示词、素材、工作流和一堆输出结果分散在各种文件夹、网页和小工具里。做久了以后，我更确定一件事：创作这件事最好有一个自己的工作区，打开就能接着上次继续，而不是每次都重新找东西、重新理流程。

## 现在主要在做的事

- 图像工作区：放文生图、图生图和输出管理
- 视频工作区：处理生成任务、队列和导出流程
- 资源库：整理提示词、模板和能反复用的素材
- 节点画布：把工作流导入、导出和整理放在一起
- 快捷应用：把商品图、风格化这类高频场景单独拎出来
- 设置中心：统一管理供应商、模型、目录、语言和工作区配置

## 技术栈

- Electron
- React
- TypeScript
- Vite
- Zustand
- XYFlow
- dnd-kit

## 本地开发

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

构建桌面应用：

```bash
npm run build
```

## 我做这个项目时比较在意的几件事

- 生成和整理不要分家
- 好用的工作流要能沉淀下来，不是用完就散
- 打开软件以后，能很快进入状态
- 视觉输出、提示词资产和节点逻辑都得算“正式内容”

## 目录说明

- `src/` 应用源码
- `scripts/` 构建辅助脚本
- `build/` 打包资源
- `public/` 静态资源
